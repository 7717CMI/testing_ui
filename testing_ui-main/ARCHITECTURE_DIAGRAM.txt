│  │ ├── Automatic deduplication                                                        │   │
│  │ └── Query invalidation on mutations                                                │   │
│  └────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                             │
│  Level 2: API Route Cache (Server)                                                         │
│  ┌────────────────────────────────────────────────────────────────────────────────────┐   │
│  │ In-Memory Cache (Next.js API Routes)                                               │   │
│  │ ├── TTL: 5 minutes                                                                  │   │
│  │ ├── Stores: Catalog overview, categories                                           │   │
│  │ └── Reduces database load                                                          │   │
│  └────────────────────────────────────────────────────────────────────────────────────┘   │
